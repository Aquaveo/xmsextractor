**********************
XmsExtractor |version|
**********************

**Last Updated:** October 5, 2018

The xmsextractor library is a extraction library for other aquaveo products. This library
contains classes used to perform extraction from a grid.

Examples
========

The examples for this library can be found in the
examples folder in the `github repository <https://github.com/Aquaveo/xmsextractor>`_
