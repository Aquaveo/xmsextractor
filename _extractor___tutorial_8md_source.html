4 ## Introduction {#Intro_Extractor}
5 The purpose of this tutorial is to provide explanation on how to use the classes defined in xmsextractor to extract data at given locations on an XmUGrid with scalar values. The examples provided in this tutorial refer to test cases that are in the xmsextractor/extractor/XmUGrid2dDataExtractor.cpp and xmsextractor/extractor/XmUGrid2dPolylineDataExtractor.cpp source files.
6 
7 ## Example - Simple Location Extractor {#Example_Simple_Extractor}
8 This is the "hello world" example for using the extractor library.
9 
10 This first example shows how to extract data from a simple XmUGrid. The testing code for this example is in XmUGrid2dDataExtractorUnitTests::testCellScalarsOnly. A picture of the example is shown below. Notice that the UGrid is a simple square from (0,0) to (1,1) split into two triangle cells. The scalar data is mapped to the cells.
11 
12 ![Simple XmUGrid with 2 Triangle Cells](images/Simple_Location_Tutorial.png)
13 
14 The basic steps to extract interpolated locations from an XmUGrid and scalar values is as follows:
15 1. Create an extractor for an existing XmUGrid (call xms::XmUGrid2dDataExtractor::New).
16 2. Set scalar and activity values (call xms::XmUGrid2dDataExtractor::SetGridCellScalars or XmUGrid2dDataExtractor::SetPointCellScalars).
17 3. Set extract locations (call xms::XmUGrid2dDataExtractor::SetExtractLocations).
18 4. Extract the data (call xms::XmUGrid2dDataExtractor::ExtractData). Values that are outside of the UGrid are returned as XM_NODATA by default. The value for no data can be set using xms::XmUGrid2dDataExtractor::SetNoDataValue.

22 ## Example - Transient Location Extractor {#Example_Transient_Extractor}
23 This example shows how to use a location extractor on transient data.
24 
25 This example shows how to extract data from a XmUGrid that has transient scalar data. The testing code for this example is in XmUGrid2dDataExtractorUnitTests::testTutorial. A picture of the example is shown below. Notice that the UGrid is a 2x3 structured grid with quadrillateral cells. There are two time steps with scalar data mapped to the points.
26 
27 ![XmUGrid With Extract Locations - Time Step 1](images/Transient_Location_Tutorial1.png)
28 ![XmUGrid With Extract Locations - Time Step 2](images/Transient_Location_Tutorial2.png)
29 
30 The steps to extract interpolated locations for transient scalar values include:
31 1. Create an extractor for an XmUGrid (call xms::XmUGrid2dDataExtractor::New).
32 2. Set extract locations (call xms::XmUGrid2dDataExtractor::SetExtractLocations).
33 3. Optionally set the "no data" value for output interpolated values (xms::XmUGrid2dDataExtractor::SetNoDataValue).
34 4. Set the point scalars for the first time step (xms::XmUGrid2dDataExtractor::SetGridPointScalars).
35 5. Extract the data (call xms::XmUGrid2dDataExtractor::ExtractData).
36 6. Continue using steps 4 and 5 for remaining time steps.

40 ## Example - Transient Polyline Extractor {#Example_Polyline_Extractor}
41 This example shows how to use a polyline extractor on transient data.
42 
43 This example shows how to extract data along a polyline from a XmUGrid that has transient scalar data. The testing code for this example is in XmUGrid2dPolylineDataExtractorUnitTests::testTransientTutorial. A picture of the example is shown below. Notice that the UGrid is a 2x3 structured grid with quadrillateral cells. There are two time steps with scalar data mapped to the points.
44 
45 ![XmUGrid With Extract Polyline - Time Step 1](images/Transient_Polyline_Tutorial1.png)
46 
47 The steps to extract interpolated values along a polyline for transient scalar values include:
48 1. Create an extractor for an XmUGrid giving the mapped location of the scalar values (call xms::XmUGrid2dPolylineDataExtractor::New).
49 2. Optionally set the "no data" value. (call xms::XmUGrid2dPolylineDataExtractor::SetNoDataValue).
50 3. Set the polyline to be extracted along (call xms::XmUGrid2dPolylineDataExtractor::SetPolyline).
51 4. Optionally get the locations used for extraction along the polyline (call xms::XmUGrid2dPolylineDataExtractor::GetExtractLocations).
52 5. Set the point scalars for the first time step (call xms::XmUGrid2dPolylineDataExtractor::SetGridScalars).
53 6. Extract the data (call xms::XmUGrid2dPolylineDataExtractor::ExtractData)
54 7. Continue using steps 5 and 6 for remaining time steps.
